Программа "ElCalc" рассчитывает сумму оплаты за электроэнергию в месяц по введённым показаниям счетчика электроэнергии (СЭ) в предположении, что используется трёх-тарифный СЭ:
- первые 100 кВт*ч тарифицируются по цене 0,456 грн/(кВт*ч),
- следующие 500 кВт - по 0,789 грн/(кВт*ч),
- каждый последующий кВт*ч - по 1,479 грн/(кВт*ч).

Программа подготовлена для ознакомления с библиотекой javax.swing.JOptionPane.
